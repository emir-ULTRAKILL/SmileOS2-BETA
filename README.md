\## English



\*\*Smiley OS 2.0\*\* is an Electron-based project designed to simulate a custom terminal-themed operating system interface. Inspired by the aesthetics of modern retro-tech and the dark, industrial designs found in games like \*\*ULTRAKILL\*\*, this project aims to provide a stylized, immersive, and functional desktop environment.

# Smiley OS 2.0
> **Status:** Beta (Active Development)

This project is currently in **Beta**. New features are being added regularly. Feel free to explore, contribute, or provide feedback!
